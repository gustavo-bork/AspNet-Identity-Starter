## ASP.NET Identity Starter

This is a project using:
- **.NET**
- **Aspire** for microservices architecture 
- **Entity Framework** for ORM
- **PostgreSQL** for database
- **Identity** for user management and information security

### How to run the project (CLI)

```sh
# Install the Aspire CLI
dotnet tool install --global Aspire.Cli
# Run the project on watch mode
aspire run --watch
```
